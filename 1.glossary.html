Tags HTML - parte 1

Títulos
são definidos pelas tags <h1> até <h6>

Listas ordenadas - <ol>
cada item da lista (<li>) é organizado em sequência

Listas não-ordenadas - <ul>
cada item da lista (<li>) é organizado sem sequência específica

Detalhes de contato
podemos criar a tag <address> para definir o criador da página e forma de contato
